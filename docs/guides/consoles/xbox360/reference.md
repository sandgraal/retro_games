## The Red Ring of Death (RROD)

### What It Was

The RROD was a widespread hardware failure affecting early Xbox 360 models:

| Aspect           | Details                            |
| ---------------- | ---------------------------------- |
| **Symptom**      | 3 red lights on power ring         |
| **Cause**        | Overheating causing solder failure |
| **Affected**     | 2005-2008 models primarily         |
| **Failure Rate** | Estimated 23-54% of early units    |
| **Cost**         | $1+ billion warranty extension     |

### Avoiding RROD Today

**Safe Models:**

- Xbox 360 S (Slim) - Jasper/Valhalla chipset
- Xbox 360 E - All variants
- Late "Phat" models with Jasper motherboard

**Identifying Jasper:**

- 12.1A power draw on label
- Manufactured after late 2008
- HDMI port present

**Warning Signs:**

- Excessive fan noise
- Overheating during play
- Graphics artifacts
- Any red light warnings

## Collecting Considerations

### Disc Condition

Xbox 360 uses DVD format:

- Scratches impact playability significantly
- Games can be installed to HDD to reduce disc use
- Disc rot is possible but uncommon
- Resurfacing can restore many discs

### Backwards Compatibility

Original Xbox games on 360:

- Emulated, not native
- ~460 games supported
- Requires HDD
- Some games enhanced
- Not all games supported
