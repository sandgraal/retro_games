## Regional Variants

### Mega Drive (Japan, 1988)

**Form Factor:** Compact design with rounded edges, "High Definition Graphics" branding.

**Key Differences:**

- First region release, original design language
- Japanese cartridge shape (rounded top)
- Headphone jack with volume slider on front
- Expansion port for Mega CD underneath
- "16-BIT" prominently displayed

**Hardware Revisions:**

1. **HAA-2510** (1988): Original Japanese model
2. **HAA-2512** (1990): Cost-reduced revision

**Collector Notes:**

- Japanese exclusives include many shooters and RPGs
- Original controller has distinct button feel
- Region-locked, requires modification for imports

### Genesis Model 1 (North America, 1989)

**Form Factor:** Same as Japanese Mega Drive, "Genesis" branding.

**Key Differences:**

- "HIGH DEFINITION GRAPHICS" label
- Larger "16-BIT" text
- "Genesis" logo on top
- Headphone jack with volume control
- Compatible with 32X and Sega CD add-ons

**Sub-Revisions:**

1. **1601** (1989): "HD Graphics" text, high-quality audio
2. **1601-05** (1990): Minor cost reductions
3. **1601-06** (1991): "TMSS" copy protection added

**Collector Notes:**

- Model 1 VA6 considered best audio quality
- Pre-TMSS units preferred by some collectors
- Sega CD compatibility on all Model 1 units

### Genesis Model 2 (North America, 1993)

**Form Factor:** Smaller, cost-reduced design.

**Key Differences:**

- Removed headphone jack
- Smaller footprint
- Different cartridge slot design
- "Sega Genesis II" text
- Mono audio output (some revisions)

**Collector Notes:**

- Works with Model 2 Sega CD and 32X
- Audio quality varies by revision
- More reliable than some Model 1 units
- Budget-friendly collector option

### Genesis Model 3 (North America, 1998)

**Form Factor:** Ultra-compact, Majesco-manufactured.

**Key Differences:**

- No expansion port (incompatible with Sega CD, 32X)
- Smallest form factor
- Made by Majesco under license
- Basic AV output only

**Collector Notes:**

- Cannot use any add-ons
- Some compatibility issues with specific games
- Budget entry point for Genesis gaming

### Mega Drive II (Europe/Japan, 1993)

**Form Factor:** Same as Genesis Model 2, regional branding.

**Key Differences:**

- PAL timing (50Hz) in European version
- "Mega Drive II" branding
- Same technical changes as Genesis Model 2

## Technical Specifications

### CPU: Motorola 68000

- **Architecture:** 32-bit internal, 16-bit external data bus
- **Clock Speed:** 7.6 MHz (NTSC), 7.5 MHz (PAL)
- **Addressing:** 24-bit (16MB address space)
- **Secondary CPU:** Zilog Z80 @ 3.58 MHz (sound/legacy)

### VDP (Video Display Processor): Yamaha YM7101

- **Resolution:** 320×224 (NTSC) or 256×224, 320×240 (PAL)
- **Colors:** 512 color palette, 61 on-screen (64 with tricks)
- **Sprites:** 80 sprites, 20 per scanline, various sizes up to 32×32
- **Scroll Planes:** 2 background layers + sprite layer
- **VRAM:** 64KB

### Audio

- **FM Synthesis:** Yamaha YM2612 (6 channels)
- **PSG:** SN76489 (4 channels, for backwards compatibility)
- **DAC:** 1 channel (uses FM channel 6)
- **Total:** Up to 10 simultaneous sound channels

**Notable Audio Features:**

- FM synthesis creates distinctive "Sega sound"
- Some games use DAC for voice samples
- Z80 handles sound processing independently

## Add-Ons & Expansions

### Sega CD (Mega-CD)

| Model      | Region | Notes                                    |
| ---------- | ------ | ---------------------------------------- |
| Mega-CD    | JP/EU  | Original front-loading design            |
| Sega CD    | NA     | Same as Mega-CD, regional branding       |
| Mega-CD II | JP/EU  | Side-loading, fits under Genesis Model 2 |
| Sega CD 2  | NA     | Side-loading, budget design              |

**Specifications:**

- CD-ROM drive (1x speed)
- Additional 1MB RAM, 512KB PCM audio RAM
- Ricoh PCM chip (8 channels)
- ASIC graphics chip with scaling/rotation

**Notable Games:** Sonic CD, Lunar, Snatcher, Sewer Shark, Night Trap

### 32X

**Purpose:** 32-bit upgrade, bridge to Saturn.

**Specifications:**

- Dual SH-2 processors @ 23 MHz
- 256KB RAM, 256KB VRAM
- 32,768 color palette
- Fits into cartridge slot, requires Genesis to function

**Collector Notes:**

- Requires multiple power adapters ("Tower of Power" setup)
- Small library (~40 games)
- Some games require both 32X AND Sega CD
- Considered a commercial failure, now collectible

### Power Base Converter

**Purpose:** Play Sega Master System games on Genesis.

**Compatibility:**

- Works with Model 1 Genesis
- Most Model 2 units (check compatibility)
- Does NOT work with Model 3 or CDX

## Cartridge Information

### Regional Formats

| Region        | Shape       | Notes                                |
| ------------- | ----------- | ------------------------------------ |
| Japan         | Rounded top | Smallest, distinct shape             |
| North America | Gridded top | Larger, distinctive ribbed design    |
| Europe        | Rounded top | Similar to Japanese, different label |

### Region Locking

- Hardware region lock based on cartridge shape
- Software region check on some games
- Easy to bypass with adapters or console modification

### Special Cartridges

- **Sonic & Knuckles:** Lock-on technology, connects to Sonic 2/3
- **EA Sports:** Yellow tab design, distinctive branding
- **Codemasters:** Unlicensed, special mapper requirements

## Collecting Guide Quick Reference

### Condition Grading

| Grade     | Box  | Manual | Cart | Expected Value |
| --------- | ---- | ------ | ---- | -------------- |
| Sealed    | Mint | N/A    | N/A  | 250-400% CIB   |
| CIB Mint  | 9-10 | 9-10   | 9-10 | 100% baseline  |
| CIB Good  | 7-8  | 7-8    | 8-9  | 60-80% CIB     |
| Complete  | 5-6  | 5-6    | 7-8  | 40-60% CIB     |
| Cart Only | N/A  | N/A    | 7-9  | 25-45% CIB     |

### What to Check When Buying

1. **Cartridge pins:** Check for corrosion, bent contacts
2. **Label condition:** Genesis labels prone to wear on edges
3. **Box type:** Cardboard (early) vs. plastic clamshell
4. **Region matching:** Cart should match box region
5. **Authenticity:** Check board revision, label quality

### Storage Recommendations

- Use Universal Game Cases for consistent display
- Store vertically to prevent warping
- Protect cardboard boxes in plastic sleeves
- Climate control important for cartridge contacts
- Keep add-ons (Sega CD, 32X) stored carefully

## Maintenance & Repair

### Common Issues

| Problem               | Cause                | Solution                          |
| --------------------- | -------------------- | --------------------------------- |
| No power              | Faulty power supply  | Replace with known-good adapter   |
| No video              | Bad AV cable or port | Try different cable, check solder |
| Sound issues          | Capacitor failure    | Recap audio section               |
| Cartridge not reading | Dirty contacts       | Clean slot and cartridge          |
| Lines on screen       | VRAM or VDP failure  | Advanced repair needed            |

### Cleaning Guide

**Cartridges:**

1. Use 91%+ isopropyl alcohol
2. Clean contacts thoroughly
3. Check for battery corrosion (games with saves)

**Console:**

1. Disassemble carefully (security screws required)
2. Clean cartridge slot contacts
3. Check capacitors for leakage (common issue)
4. Consider full recap on older units
