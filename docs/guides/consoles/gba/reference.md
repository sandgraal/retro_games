## Technical Specifications

### CPU

| Component        | Specification                       |
| ---------------- | ----------------------------------- |
| **Main CPU**     | ARM7TDMI @ 16.78 MHz                |
| **Co-processor** | Z80 @ 4.19 / 8.38 MHz (GB/GBC mode) |
| **Architecture** | 32-bit RISC (ARM)                   |
| **Performance**  | ~16.78 MIPS                         |

### Memory

| Type           | Specification               |
| -------------- | --------------------------- |
| **System RAM** | 32KB + 96KB VRAM            |
| **External**   | 256KB (work RAM)            |
| **Cartridge**  | Up to 32MB ROM + SRAM/Flash |

### Graphics

| Component       | Specification                     |
| --------------- | --------------------------------- |
| **Resolution**  | 240 × 160 pixels                  |
| **Colors**      | 32,768 (15-bit)                   |
| **On-screen**   | 512 colors in "character" mode    |
| **Backgrounds** | 4 layers                          |
| **Sprites**     | 128 sprites, 8×8 to 64×64         |
| **Modes**       | Modes 0-5 (tile and bitmap modes) |
| **Effects**     | Rotation, scaling, alpha blending |

### Display Modes

| Mode   | Type   | BG Layers | Features                    |
| ------ | ------ | --------- | --------------------------- |
| Mode 0 | Tile   | 4         | Standard tile mode          |
| Mode 1 | Tile   | 3         | One affine (rotation) BG    |
| Mode 2 | Tile   | 2         | Both affine BGs             |
| Mode 3 | Bitmap | 1         | 240×160 @ 15-bit color      |
| Mode 4 | Bitmap | 2         | 240×160 @ 8-bit, page flip  |
| Mode 5 | Bitmap | 2         | 160×128 @ 15-bit, page flip |

### Audio

| Component    | Specification                              |
| ------------ | ------------------------------------------ |
| **Channels** | 6 total (2 PWM + 4 legacy GB)              |
| **PWM DAC**  | Dual 8-bit DAC channels                    |
| **Legacy**   | 4 Game Boy channels (2 pulse, wave, noise) |
| **Output**   | Stereo headphone jack, mono speaker        |

### Physical (Original GBA)

| Dimension     | Specification               |
| ------------- | --------------------------- |
| **Size**      | 144.5 × 82 × 24.5 mm        |
| **Weight**    | 140g (without batteries)    |
| **Screen**    | 2.9" TFT LCD (no backlight) |
| **Power**     | 2× AA batteries (15 hours)  |
| **Link Port** | Proprietary serial          |

## Hardware Variants

### Game Boy Advance (AGB-001) - 2001

**Original Model:**

| Feature         | Details                             |
| --------------- | ----------------------------------- |
| **Form Factor** | Horizontal "landscape" design       |
| **Screen**      | 2.9" TFT, NO backlight              |
| **Power**       | 2× AA (15 hours)                    |
| **Colors**      | Purple, glacier, various            |
| **Issues**      | Very difficult to see without light |

**Collector Notes:**

- Original design, nostalgic
- Screen visibility is problematic
- Many color variants exist
- Backlight mods popular

### Game Boy Advance SP (AGS-001) - 2003

**Clamshell Redesign:**

| Feature         | Details                             |
| --------------- | ----------------------------------- |
| **Form Factor** | Clamshell/flip design               |
| **Screen**      | Frontlit TFT LCD                    |
| **Power**       | Internal lithium-ion (10+ hours)    |
| **Charger**     | Proprietary AC adapter              |
| **Colors**      | Silver, onyx, many special editions |

**Collector Notes:**

- Much better screen than original
- Compact, protected screen
- No headphone jack (adapter required)
- Still frontlit, not backlit

### Game Boy Advance SP (AGS-101) - 2005

**Backlit Upgrade:**

| Feature          | Details                  |
| ---------------- | ------------------------ |
| **Form Factor**  | Same as AGS-001          |
| **Screen**       | **Backlit** TFT LCD      |
| **Brightness**   | Far superior to AGS-001  |
| **Availability** | Less common than AGS-001 |

**Collector Notes:**

- **THE** version to own for playing
- Significantly better screen
- Commands premium price ($100-150+)
- Pearl pink and graphite are common colors
- Can identify by brighter screen or model number

### Game Boy Micro - 2005

**Ultra-Compact Design:**

| Feature           | Details                    |
| ----------------- | -------------------------- |
| **Form Factor**   | Tiny horizontal design     |
| **Screen**        | 2" backlit LCD             |
| **Size**          | 101 × 50 × 17.2 mm         |
| **Weight**        | 80g                        |
| **Compatibility** | GBA games ONLY (no GB/GBC) |

**Collector Notes:**

- Very collectible, premium prices
- Beautiful tiny design
- Faceplates are interchangeable
- No backwards compatibility
- $150-300+ depending on condition

## Collecting Tips

### What to Look For

**Console Condition:**

- Screen scratches (common on SP)
- Battery contacts (corrosion on original)
- Button responsiveness
- Hinge condition (SP)
- Shell cracks or damage

**Cartridge Condition:**

- Label quality
- Shell integrity
- Authentic vs reproduction
- Battery status (save games)

### Authentication

GBA fakes are extremely common:

**Signs of Fake Carts:**

- Wrong Nintendo logo font
- Poor label print quality
- Incorrect cart color
- Wrong screw type (not tri-wing)
- Light cart weight
- No indentation on cart back

**Most Commonly Faked:**

- All Pokemon games
- Fire Emblem games
- Metroid games
- Mother 3
- Any expensive game

### Screen Modding

Popular mods for original GBA:

- IPS screen kits (backlit)
- AGS-101 screen transplant
- Rechargeable battery mods
- New shell replacements
