# Collecting for Game Boy Advance: The Complete Guide

_Last updated: December 2025_

## Introduction

The Game Boy Advance represented the pinnacle of 2D gaming, packing SNES-quality graphics into a portable form factor. Released in 2001, the GBA library is packed with original masterpieces, excellent ports, and some of the best RPGs ever made. For collectors, the GBA offers a fantastic blend of quality games and the challenges of authentication in a market flooded with counterfeits.

**Why Collect for GBA?**

- Arguably the best 2D game library ever assembled
- Portable SNES-quality gaming
- Excellent original games AND ports
- Fire Emblem, Metroid, Pokemon at their portable best
- Multiple hardware variants to collect
- Active modding scene for display upgrades

**Current Market Position:** GBA collecting is competitive. Popular games command strong prices, and counterfeit cartridges are epidemic. Authentication skills are essential.

## Console Buying Guide

### Which Model to Buy?

| Model          | Screen      | Pros                         | Cons                       | Price     |
| -------------- | ----------- | ---------------------------- | -------------------------- | --------- |
| GBA (AGB-001)  | Unlit       | Classic design, moddable     | Can't see screen           | $40-60    |
| SP AGS-001     | Frontlit    | Compact, lit screen          | Dim compared to 101        | $60-100   |
| **SP AGS-101** | **Backlit** | **Best screen for playing**  | Pricier, less common       | $100-150+ |
| Game Boy Micro | Backlit     | Tiny, beautiful, collectible | No GB/GBC, faceplates rare | $150-300+ |

**Recommendation:** For playing, get an AGS-101 SP. For collecting, the original GBA with IPS mod is popular. For display, the Micro is gorgeous.

### Identifying AGS-101 vs AGS-001

The AGS-101 has a dramatically brighter screen:

| Check Method      | AGS-001 (Frontlit)   | AGS-101 (Backlit)    |
| ----------------- | -------------------- | -------------------- |
| Model number      | AGS-001 on sticker   | AGS-101 on sticker   |
| Screen brightness | Somewhat dim         | Noticeably bright    |
| Screen uniformity | Light bleed on edges | Even illumination    |
| Common colors     | Silver, onyx, etc.   | Pearl pink, graphite |
| Price             | Lower                | Higher               |

## Fake Cartridge Detection

### The Counterfeit Problem

GBA fakes are **extremely common**. Estimates suggest 30-50% of online GBA listings are counterfeits.

### How to Spot Fakes

**Visual Inspection:**

| Element        | Authentic                               | Fake                         |
| -------------- | --------------------------------------- | ---------------------------- |
| Nintendo logo  | Crisp, correct font                     | Blurry, wrong font           |
| Label printing | High quality, correct colors            | Faded, wrong colors          |
| Cart color     | Matches game (clear red for Pokemon FR) | Wrong shade or material      |
| Cart back      | Has Nintendo indentation                | Missing or wrong indentation |
| Screw          | Tri-wing security screw                 | Phillips or wrong tri-wing   |
| Weight         | Solid, substantial                      | Light, hollow feeling        |

**Opening the Cart (Definitive):**

| Element         | Authentic                         | Fake                           |
| --------------- | --------------------------------- | ------------------------------ |
| PCB quality     | Professional, "NINTENDO" on board | Amateur, no Nintendo marking   |
| Chip labels     | Printed, professional             | Hand-written or sticker labels |
| Battery (saves) | Tab-welded battery                | Battery holder or missing      |
| Component count | Clean, minimal chips              | Blob chips, extra components   |
| Gold contacts   | Shiny, even                       | Dull, uneven                   |

### Most Commonly Faked Games

1. **ALL Pokemon games** (90%+ of cheap listings are fake)
2. Fire Emblem games
3. Metroid games
4. Castlevania: Aria of Sorrow
5. Mother 3
6. Golden Sun games
7. Any game over $50

### Where to Buy Authentic

| Source                  | Risk Level | Notes                         |
| ----------------------- | ---------- | ----------------------------- |
| Local game stores       | Low        | Can inspect before buying     |
| eBay (verified sellers) | Medium     | Check photos carefully        |
| r/GameSale              | Medium     | Community, can request photos |
| Amazon                  | **HIGH**   | Commingled inventory, avoid   |
| Facebook Marketplace    | Medium     | Inspect in person             |
| Mercari                 | Medium     | Request board photos          |

**Always request photos of the cartridge board before buying expensive games online.**

## Collecting Tips

### What to Look For

**Console Condition:**

- Screen scratches (SP closes, should be clean)
- Button responsiveness
- Battery contacts (AAs corrode)
- Hinge tightness (SP)
- Shell cracks
- Link port condition

**Cartridge Condition:**

- Label damage (fading, tears, stickers)
- Shell damage
- Pin corrosion
- Save battery health (test saves!)

### Save Battery Replacement

Many GBA carts use CR1616 batteries:

**Games with Save Batteries:**

- Pokemon Ruby/Sapphire/Emerald (clock feature)
- Most RPGs with save files
- Some older ports

**Battery Life:** ~10-15 years (many need replacement now)

**Note:** Pokemon R/S/E battery only affects clock, not save files.

### Storage & Preservation

**Cartridges:**

- Store in protective cases or boxes
- Keep dry and cool
- Clean pins with isopropyl alcohol
- Consider replacing save batteries proactively

**Consoles:**

- Remove batteries when not in use (original GBA)
- Keep SP charged occasionally to maintain battery
- Store in cases to prevent scratches
- Keep chargers (proprietary for SP/Micro)
