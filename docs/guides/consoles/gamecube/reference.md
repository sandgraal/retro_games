## Regional Variants

### Japanese GameCube (2001)

**Distinguishing Features:**

- First region release, September 14, 2001
- Violet/Indigo as primary launch color
- NTSC-J format
- 283 official releases
- Many exclusive titles and variants

**Notable Exclusives:**

- Doshin the Giant (later PAL)
- GiFTPiA
- Homeland
- Various visual novels and anime games

### North American GameCube (2001)

**Distinguishing Features:**

- Launched November 18, 2001
- Indigo primary color at launch
- NTSC-U format
- 299 official releases
- Best-selling region for GameCube

**Launch Titles:**

- Luigi's Mansion
- Star Wars Rogue Squadron II
- Wave Race: Blue Storm
- Super Monkey Ball
- Tony Hawk's Pro Skater 3

### European GameCube (2002)

**Distinguishing Features:**

- Launched May 3, 2002
- PAL format (50/60Hz options in many games)
- 327 official releases
- Some titles modified for European ratings

**Regional Notes:**

- Delayed release hurt market position
- Many games include 60Hz mode
- Some exclusive versions

### DOL-001 vs DOL-101

**DOL-001 (Original):**

- Digital AV Out port (component cable compatible)
- Serial Port 2 (bottom)
- Preferred by collectors for video quality

**DOL-101 (Cost-reduced):**

- No Digital AV Out (component via adapter only)
- No Serial Port 2
- Lower collector demand

### Disc Preservation

GameCube discs are mini DVDs and relatively durable:

- Store in original cases
- Avoid scratches on data side
- Professional resurfacing works well
- Disc rot is rare but possible on early pressings

## Maintenance and Repair

### Common Issues

| Issue                  | Cause                  | Solution                |
| ---------------------- | ---------------------- | ----------------------- |
| Disc read errors       | Dirty lens, worn motor | Clean lens, adjust POT  |
| No power               | Fuse, power supply     | Check fuse, replace PSU |
| Controller port issues | Dirty contacts         | Clean ports, resolder   |
| Video problems         | AV port, cable issues  | Clean port, test cables |
| Fan noise              | Dust, failing fan      | Clean or replace fan    |

### Disc Drive Maintenance

1. **Cleaning**: Use lens cleaning disc or manual cleaning
2. **POT Adjustment**: Laser power calibration (advanced)
3. **Replacement**: Drives still available
4. **Note**: Later revisions may have compatibility issues

### Optical Drive Emulators (ODE)

| Device    | Features                              |
| --------- | ------------------------------------- |
| GC Loader | SD card loading, popular choice       |
| Swiss     | Homebrew loader for ODE               |
| SD2SP2    | Memory card slot SD adapter (not ODE) |

## Collecting Strategy

### Starter Collection (~$300-$500)

1. Console with controller and cables
2. Memory Card 251 or larger
3. Core essentials: Melee, Wind Waker, Metroid Prime
4. One Mario game: Sunshine or Mario Kart
5. One multiplayer: Mario Party or TimeSplitters

### Advanced Collection (~$2,000-$4,000)

- Complete first-party Nintendo library
- All Resident Evil games
- Premium RPGs (Tales, Baten Kaitos, Skies of Arcadia)
- Component cable or quality HDMI adapter
- WaveBird controller set

### Complete Set

- **NA Complete**: 299 games, ~$25,000-$40,000
- **Timeline**: 4-8 years typical
- **Challenge Games**: Fire Emblem, Pokémon Box, Gotcha Force
