## Console Overview

### Introduction

The PlayStation Portable (PSP) was Sony's first handheld console, launched to compete directly with Nintendo's Game Boy and DS dominance. It represented an unprecedented leap in portable gaming power, bringing console-quality graphics and multimedia capabilities to a handheld device. The PSP was as much a media player as a gaming system, capable of playing movies, music, and displaying photos.

### Historical Significance

**Industry Impact:**

- First handheld with console-quality 3D graphics
- Established Sony in the portable gaming market
- Pioneered multimedia convergence in handhelds
- UMD format for games and movies (unique proprietary format)
- Influenced smartphone gaming development
- Created robust modding/homebrew scene

**Technical Achievements:**

- First 16:9 widescreen handheld display
- Full-speed PS1 backward compatibility (via downloads)
- Remote Play with PlayStation 3
- LocationFree TV integration
- Custom firmware scene rivaling any in gaming history

## Hardware Variants

### PSP-1000 (Original / "Phat")

| Attribute          | Specification                               |
| ------------------ | ------------------------------------------- |
| **Release**        | December 2004 (JP)                          |
| **Weight**         | 280g with battery                           |
| **Battery**        | 1800 mAh (3-6 hours)                        |
| **RAM**            | 32 MB                                       |
| **Screen**         | 4.3" TFT LCD                                |
| **UMD Drive**      | Standard speed                              |
| **Video Out**      | No                                          |
| **Colors**         | Piano Black, Ceramic White, various limited |
| **Collectibility** | ★★★☆☆ Common                                |

**Notable Characteristics:**

- Heavier build quality with substantial feel
- Screen issues: ghosting on fast motion, dead pixels common
- IrDA port for limited peripheral support
- More prone to UMD drive failures
- Most limited edition variants released on this model

### PSP-2000 (Slim & Lite)

| Attribute          | Specification                       |
| ------------------ | ----------------------------------- |
| **Release**        | September 2007 (worldwide)          |
| **Weight**         | 189g with battery (33% lighter)     |
| **Battery**        | 1200 mAh (4-6 hours)                |
| **RAM**            | 64 MB (doubled)                     |
| **Screen**         | 4.3" TFT LCD (improved response)    |
| **UMD Drive**      | Faster read speeds                  |
| **Video Out**      | Component, S-Video, composite       |
| **Colors**         | Multiple including metallic options |
| **Collectibility** | ★★★★☆ Ideal balance                 |

**Notable Characteristics:**

- 33% thinner profile than PSP-1000
- TV output capability (via AV cables)
- Faster loading times for games
- Reduced ghosting on display
- Better homebrew compatibility (early motherboards)
- Most versatile model for modern use

### PSP-3000 (Brite)

| Attribute          | Specification                           |
| ------------------ | --------------------------------------- |

| **RAM**            | 64 MB                                   |

**Notable Characteristics:**

- Best native screen quality of any PSP
- Built-in microphone for Skype/voice chat
- Wider color gamut and viewing angles
- Scanline issue on video output (interlaced only)
- More resistant to custom firmware initially
- Premium limited editions (Monster Hunter, etc.)

### PSP Go (PSP-N1000)

| Attribute          | Specification             |
| ------------------ | ------------------------- |

| **Weight**         | 158g                      |
| **Form Factor**    | Sliding screen design     |

- 16 GB internal storage + Memory Stick Micro slot
- Failed commercially but beloved by collectors
- Pause/Resume feature for any game
- Premium pricing limits collection value

## Regional Variants

### Japanese Systems (NTSC-J)

- First to receive new models and colors
- Exclusive limited editions (Monster Hunter, FF, etc.)
- 100V power adapter
- Japanese language default (English available)
- Best selection of exclusive games
- Import-friendly for most games

### North American Systems (NTSC-U/C)

- Standard North American voltage
- English language default
- Largest English game library
- Value Pack editions with accessories
- Entertainment Packs with bundled games

### European Systems (PAL)

- 220V power adapter
- Multi-language support
- Received PSP Street exclusively
- Extended support timeline
- Platinum/Essentials budget releases

### Asian Systems

- Often region-free
- Multiple language options
- Unique color variants
- Lower pricing in some markets
- Mixed game availability

## Technical Specifications

### Processor Details

**CPU: MIPS R4000-based "Allegrex"**

- Variable clock: 1-333 MHz (user adjustable in later firmware)
- Originally limited to 222 MHz for battery life
- 128-bit VFPU for vector math
- Full MIPS III instruction set

**GPU: "Graphics Engine"**

- 166 MHz clock speed
- 2 million polygons/second
- Hardware T&L
- Compressed textures
- 33 million pixels/second fill rate

### Memory Architecture

| Type       | PSP-1000   | PSP-2000+  |
| ---------- | ---------- | ---------- |
| Main RAM   | 32 MB      | 64 MB      |
| eDRAM      | 2 MB (GPU) | 2 MB (GPU) |
| NAND Flash | 32 MB      | 32 MB      |

## Media Formats

### UMD (Universal Media Disc)

| Attribute      | Specification             |
| -------------- | ------------------------- |
| **Capacity**   | 1.8 GB (dual-layer)       |
| **Diameter**   | 60mm                      |
| **Type**       | Optical disc in cartridge |
| **Read Speed** | Up to 11 Mbps             |

**UMD Types:**

- **UMD-Game**: Games and interactive content
- **UMD-Video**: Movies and video content
- **UMD-Music**: Audio content (rare)
- **UMD-PG**: PlayStation Portable game (standard)

### Memory Storage

**Memory Stick Pro Duo:**

- 32 MB to 32 GB capacity
- Required for saves, downloads, media
- Third-party adapters for MicroSD cards
- PSP Go uses Memory Stick Micro (M2)

## Common Issues & Repairs

### UMD Drive Problems

**Symptoms:** Disc not reading, grinding noise, eject issues

**Solutions:**

1. Clean disc and lens with isopropyl alcohol
2. Replace UMD drive (model-specific, $15-30)
3. Check ribbon cable connections
4. Lubricate drive mechanism

**Prevention:** Store UMDs in cases, avoid dust

### Analog Stick Drift

**Symptoms:** Character movement without input, dead zones

**Solutions:**

1. Clean around stick with compressed air
2. Replace analog stick module ($5-10)
3. Recalibrate in system settings

**Difficulty:** Easy-moderate (requires opening console)

### Battery Issues

**Symptoms:** Won't hold charge, swollen battery, no power

**Solutions:**

1. Replace battery (official hard to find, aftermarket $10-20)
2. Check battery contacts for corrosion
3. Test with known-good battery

**Warning:** Swollen batteries are dangerous - dispose properly

### Screen Problems

**Symptoms:** Dead pixels, backlight bleeding, display lines

**Solutions:**

1. Dead pixels: Software pixel fixers rarely work
2. Screen replacement ($20-40) for major issues
3. Check ribbon cable for display lines

**Note:** PSP-3000 screens have different connectors than earlier models

### Power/Charging Issues

**Symptoms:** Won't charge, turns off randomly, no power

**Solutions:**

1. Try different AC adapter
2. Clean charge port with alcohol
3. Replace internal fuse (advanced)
4. Check power switch flex cable

## Collecting Tips

### Best Entry Points

1. **Budget Start:** PSP-2000 + popular games
2. **Best Screen:** PSP-3000 for display quality
3. **Digital Focus:** PSP Go if disc-free appeals
4. **Complete Set:** One of each variant

### What to Look For

**Console Purchase:**

- Test UMD drive with known working disc
- Check analog stick for drift
- Verify all buttons function
- Inspect screen for dead pixels
- Test Wi-Fi connectivity
- Check battery health

**Game Purchase:**

- Verify UMD disc condition (scratches affect reading)
- Check case for cracks (common on PSP cases)
- Confirm manual presence
- Look for complete insert materials

### Storage Recommendations

**Console Storage:**

- Remove battery for long-term storage
- Store in protective case
- Avoid extreme temperatures
- Keep away from direct sunlight

**Game Storage:**

- Store UMDs vertically in cases
- Use protective sleeves for loose discs
- Climate-controlled environment
- Keep away from magnets (minimal risk but precaution)

## Emulation & Preservation

### Official Options

**PlayStation Store (Discontinued):**

- Store closed 2021 (purchases preserved)
- No new purchases possible
- Downloaded games playable indefinitely

**PlayStation Plus (Limited):**

- Some PSP classics on PS4/PS5
- Streaming only for some titles
- Limited library

### Emulation Scene

**PPSSPP:**

- Leading PSP emulator
- Available on PC, Mac, Linux, Android, iOS
- High compatibility (most games playable)
- Enhancement features (upscaling, texture replacement)
- Save state support
- Netplay for multiplayer

**Compatibility Considerations:**

- Most games run at full speed on modern devices
- Some games have specific issues (check compatibility lists)
- 2x-10x internal resolution scaling possible
- Vulkan backend for best performance
