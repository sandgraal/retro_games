## Regional Variants

### Famicom (Japan, 1983)

**Form Factor:** Compact design with hardwired controllers, top-loading cartridge slot.

**Key Differences:**

- 60-pin cartridge connector (not compatible with NES games without adapter)
- Controllers permanently attached, Player 2 controller has microphone
- Expansion port on front for accessories
- No lockout chip (10NES)
- Red and white color scheme

**Collector Notes:**

- Original white/red is most common; later "New Famicom" (AV Famicom) released 1993
- AV Famicom uses standard AV cables vs. RF on original
- Many exclusive games never released outside Japan

### NES (North America, 1985)

**Form Factor:** VCR-style front-loading design, detachable controllers.

**Key Differences:**

- 72-pin cartridge connector (infamous for bending/corrosion issues)
- 10NES lockout chip (region locking, blinking light issues)
- Controllers detachable with 7-foot cables
- Gray "toaster" design

**Hardware Revisions:**

1. **NES-001** (1985): Original front-loader, most common
2. **NES-101** (1993): "Top Loader" redesign, improved reliability, no AV output (RF only on US model)

**Collector Notes:**

- Top Loader highly sought after for reliability
- Action Set (with Zapper, SMB/Duck Hunt) most common bundle
- Deluxe Set with R.O.B. robot is rare collector piece

### NES (Europe/PAL, 1986)

**Form Factor:** Same as North American NES-001.

**Key Differences:**

- PAL timing (50Hz vs 60Hz) means slightly slower gameplay
- Different lockout chip regions (PAL-A for UK/Italy/Australia, PAL-B for most of Europe)
- Many games have borders due to resolution differences
- Some games run 17.5% slower than NTSC versions

**Collector Notes:**

- PAL games often cheaper than NTSC equivalents
- Some collectors prefer NTSC for authentic speed
- Australian releases particularly uncommon

## Technical Specifications

### CPU: Ricoh 2A03

- **Architecture:** Modified MOS 6502
- **Clock Speed:** 1.79 MHz (NTSC), 1.66 MHz (PAL)
- **Features:** Built-in audio processing unit (APU)
- **RAM:** 2KB internal

### PPU (Picture Processing Unit): Ricoh 2C02

- **Resolution:** 256×240 pixels (NTSC displays 256×224)
- **Colors:** 54-color palette, 25 colors on-screen simultaneously
- **Sprites:** 64 sprites, 8 per scanline, 8×8 or 8×16 pixels
- **Backgrounds:** 2 nametables (scrolling support)
- **VRAM:** 2KB internal + cartridge ROM/RAM

### Audio

- **Channels:** 5 total
  - 2× Pulse wave (variable duty cycle)
  - 1× Triangle wave
  - 1× Noise generator
  - 1× DPCM (sample playback)
- **Expansion Audio:** Famicom cartridges could add audio chips (VRC6, VRC7, N163, FDS, MMC5, Sunsoft 5B)

## Cartridge Information

### Physical Formats

| Format  | Pins | Region   | Notes                               |
| ------- | ---- | -------- | ----------------------------------- |
| Famicom | 60   | Japan    | Smaller cartridge, many colors      |
| NES     | 72   | NA/EU/AU | Larger "brick" style, gray standard |

### Special Cartridge Types

- **Gold cartridges:** Zelda I, Zelda II (NA) - highly collectible
- **Black cartridges:** Unlicensed games (Tengen, Color Dreams, etc.)
- **Blue cartridges:** Unlicensed (Camerica, Codemasters)
- **Licensed variants:** Some games have label/cart variations

### Mapper Chips

Games used various mapper chips for expanded capabilities:

| Mapper | Name | Features                       | Notable Games                   |
| ------ | ---- | ------------------------------ | ------------------------------- |
| 0      | NROM | No mapper (32KB PRG, 8KB CHR)  | Super Mario Bros., Donkey Kong  |
| 1      | MMC1 | Bank switching, battery saves  | Zelda, Metroid, Final Fantasy   |
| 4      | MMC3 | IRQ timer, more banks          | Super Mario Bros. 3, Mega Man   |
| 5      | MMC5 | Most advanced, expansion audio | Castlevania III, Laser Invasion |

## Collecting Guide Quick Reference

### Condition Grading

| Grade     | Box  | Manual | Cart | Expected Value |
| --------- | ---- | ------ | ---- | -------------- |
| Sealed    | Mint | N/A    | N/A  | 300-500% CIB   |
| CIB Mint  | 9-10 | 9-10   | 9-10 | 100% baseline  |
| CIB Good  | 7-8  | 7-8    | 8-9  | 60-80% CIB     |
| Complete  | 5-6  | 5-6    | 7-8  | 40-60% CIB     |
| Cart Only | N/A  | N/A    | 7-9  | 20-40% CIB     |

### What to Check When Buying

1. **72-pin connector:** Test in known-working console
2. **Battery games:** Check if save battery still works (Zelda, Final Fantasy, etc.)
3. **Label condition:** Fading, tears, marker damage
4. **Cart shell:** Cracks, yellowing, screw holes stripped
5. **Authenticity:** Weight, label font, board inspection for reproductions

### Storage Recommendations

- Store cartridges vertically in protective cases
- Climate control: 60-70°F, 30-50% humidity
- Keep boxes in comic book bags with acid-free boards
- Use cartridge protectors for loose carts
- Avoid direct sunlight (label fading, yellowing)

## Maintenance & Repair

### Common Issues

| Problem              | Cause                    | Solution                         |
| -------------------- | ------------------------ | -------------------------------- |
| Blinking power light | 72-pin connector worn    | Clean or replace connector       |
| No video output      | AV cable or PPU failure  | Test cables, may need PPU repair |
| Game resets randomly | Dirty cartridge contacts | Clean with isopropyl alcohol     |
| Dead save battery    | CR2032 depleted          | Replace battery (soldering req.) |
| Audio crackling      | Dirty expansion port     | Clean contacts, check capacitors |

### Cleaning Guide

**Cartridges:**

1. Use 91%+ isopropyl alcohol on cotton swab
2. Gently clean contacts until no residue on swab
3. Let dry completely before inserting

**Console (72-pin connector):**

1. Disassemble console (requires security bit)
2. Gently bend pins back to original position
3. Clean with isopropyl alcohol
4. Consider boiling method for deep cleaning
5. Replacement connectors available (~$8-15)
