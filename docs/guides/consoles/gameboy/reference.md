## Console Overview

### Introduction

The Game Boy family represents Nintendo's dominance in handheld gaming across three generations. From the original 1989 Game Boy through the final Game Boy Advance SP, these systems defined portable gaming and created one of the largest combined game libraries in history.

### Historical Significance

**Game Boy (1989):**

- Revolutionized portable gaming
- Outlasted technologically superior competitors (Game Gear, Lynx)
- Tetris bundle drove mainstream adoption
- Survived 10+ years with minimal updates

**Game Boy Color (1998):**

- Full color while maintaining backward compatibility
- Extended Game Boy's lifespan against emerging competitors
- Pokémon Gold/Silver became system sellers

**Game Boy Advance (2001):**

- 32-bit handheld rivaling SNES capabilities
- Home console ports became viable
- GBA SP redesign became iconic
- Final cartridge-based Nintendo handheld

### Form Factors

**Original Game Boy (DMG-01):**

- Large, bulky design ("brick")
- Requires 4 AA batteries
- No backlight, pea-green screen
- External link cable port

**Game Boy Pocket (MGB-001):**

- 30% smaller than original
- Improved screen (gray instead of green)
- 2 AAA batteries
- Thinner link cable

**Game Boy Light (MGB-101):**

- Japan-only release (1998)
- Built-in electroluminescent backlight
- Uses 2 AA batteries
- Highly sought by collectors

**Game Boy Color (CGB-001):**

- Slightly larger than Pocket
- Infrared communication port
- Improved speaker
- 2 AA batteries

**Game Boy Advance (AGB-001):**

- Horizontal "landscape" design
- 2 AA batteries
- Link port + external AC adapter port
- Shoulder buttons (L/R)

**Game Boy Advance SP (AGS-001/AGS-101):**

- Clamshell design with hinge
- Built-in rechargeable battery
- Frontlit (AGS-001) or backlit (AGS-101) screen
- Integrated speaker

**Game Boy Micro (OXY-001):**

- Smallest Game Boy ever
- GBA games only (no backward compatibility)
- High-quality backlit screen
- Interchangeable faceplates

## Technical Specifications

### Game Boy / Game Boy Color

**CPU - Sharp LR35902:**

- Custom 8-bit processor (Z80-derived)
- 4.19 MHz (GB) / 8.39 MHz dual-speed (GBC)
- 256-byte internal RAM
- Not Z80 compatible despite similarities

**Graphics:**

- 160 × 144 pixel LCD
- 4 shades (GB) / 56 simultaneous colors from 32,768 (GBC)
- 8 × 8 or 8 × 16 pixel sprites
- 40 sprites total, 10 per scanline
- Background layer with scrolling
- Window layer for HUD elements

**Memory:**

- 8 KB internal RAM (GB) / 32 KB (GBC)
- 8 KB video RAM (GB) / 16 KB (GBC)
- Cartridge ROM: 32 KB to 8 MB
- Cartridge RAM: 0 to 128 KB (battery-backed saves)

**Sound:**

- 4 sound channels
- 2 square wave with sweep/envelope
- 1 4-bit wavetable
- 1 noise channel
- Stereo output through headphones

### Game Boy Advance

**CPU - ARM7TDMI:**

- 32-bit RISC processor at 16.78 MHz
- Additional Z80 core for backward compatibility
- THUMB instruction set for code density

**Graphics:**

- 240 × 160 pixel TFT LCD
- 32,768 colors (15-bit)
- Mode 0-2: Tile-based modes (similar to SNES)
- Mode 3-5: Bitmap modes (framebuffer)
- 128 sprites, 32 × 64 pixels max
- Affine transformation (rotation/scaling)
- 4 background layers

**Memory:**

- 32 KB internal WRAM
- 256 KB external WRAM
- 96 KB VRAM
- Cartridge ROM: 1 MB to 32 MB
- Cartridge RAM: 0 to 128 KB

**Sound:**

- All Game Boy sound hardware (for compatibility)
- 2 additional Direct Sound channels (8-bit PCM)
- Stereo output

## Cartridge Information

### Game Boy / Game Boy Color

**Physical Format:**

- Small gray cartridges (GB)
- Transparent colored cartridges (many GBC games)
- Black cartridges (GBC-only games)

**Cartridge Types:**

| Type        | Color       | Compatibility                   |
| ----------- | ----------- | ------------------------------- |
| Original GB | Gray        | GB, GBC, GBA (non-enhanced)     |
| Dual-mode   | Variable    | Enhanced on GBC, playable on GB |
| GBC-only    | Black/Clear | GBC and GBA only, notched cart  |

**Memory Controllers (Mappers):**

- MBC1: Up to 2MB ROM, 32KB RAM
- MBC2: 256KB ROM, 512×4 bits RAM
- MBC3: Real-time clock (Pokémon G/S)
- MBC5: Up to 8MB ROM, rumble support
- MBC7: Accelerometer (Kirby Tilt 'n' Tumble)

**Battery Backup:**

- CR2025 or CR1616 batteries for save games
- Batteries last 10-20 years typically
- Pokémon games notorious for battery death

### Game Boy Advance

**Physical Format:**

- Smaller, rectangular cartridges
- Not backward compatible with GB/GBC hardware
- Cannot play in DS Lite slot (GBA slot only)

**Features:**

- Flash ROM saves (some games)
- EEPROM or FRAM saves (no battery needed)
- Real-time clock in some games
- Rumble (Drill Dozer, some Pokémon)
- Solar sensor (Boktai series)

## Hardware Maintenance

### Common Issues

**Game Boy (DMG):**

- Vertical/horizontal screen lines (ribbon cable)
- Battery corrosion (cartridge slot)
- Speaker deterioration
- Power switch wear

**Game Boy Color:**

- Screen scratches
- Battery terminal corrosion
- Power switch issues
- Speaker problems

**Game Boy Advance:**

- Screen scratches (no protection)
- Battery compartment corrosion
- Shoulder button wear
- Link port damage

**GBA SP:**

- Hinge wear and cracking
- Battery degradation
- Charging port issues
- Screen ribbon cable stress

### Cleaning Guide

**Console Exterior:**

1. Remove batteries
2. Use isopropyl alcohol (90%+) on cotton swabs
3. Clean all buttons and crevices
4. Dry thoroughly before use

**Cartridge Slot:**

1. Use contact cleaner on cotton swab
2. Gently clean contacts
3. Allow to dry completely
4. Test with known-working game

**Screen:**

1. Use plastic-safe cleaner only
2. Soft microfiber cloth
3. Avoid pressure on LCD
4. Replace scratched lenses

### Battery Replacement

**Game Boy/GBC/GBA Cartridge Saves:**

Common batteries: CR2025, CR1616

Tools needed:

- 3.8mm gamebit screwdriver
- Soldering iron (for original games)
- Replacement battery

**GBA SP Internal Battery:**

- AGS-003 replacement battery
- Tri-wing screwdriver required
- 600-850 mAh typical

## Collecting Tips

### Authentication

**Original vs. Reproduction:**

Bootleg cartridges are extremely common. Check for:

- Label quality and printing alignment
- Nintendo seal of quality
- Correct screw type (gamebit, not Phillips)
- PCB quality (visible through cartridge)
- Battery/save chip placement
- ESRB rating consistency

**Common Bootleg Targets:**

- All Pokémon games
- The Legend of Zelda series
- Popular Nintendo first-party titles
- Valuable games (Shantae, Ninja Five-O)

### Condition Factors

**Game Boy Cartridges:**

| Condition | Description                 | Value Impact |
| --------- | --------------------------- | ------------ |
| Mint      | Perfect, like new           | +100-200%    |
| Near Mint | Minimal wear, clean label   | +50-80%      |
| Excellent | Light wear, no damage       | +20-40%      |
| Very Good | Normal wear, readable label | Baseline     |
| Good      | Moderate wear, playable     | -20-40%      |
| Fair      | Heavy wear, may have damage | -50-70%      |

**Console Grading:**

- Screen condition (scratches, dead pixels)
- Button functionality and feel
- Cosmetic wear (shell, battery door)
- Stickers, writing, modifications
- Original packaging adds significant value
