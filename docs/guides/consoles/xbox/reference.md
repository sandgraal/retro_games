## Regional Variants

### North American Xbox (2001)

**Distinguishing Features:**

- First region launch, November 15, 2001
- Largest library and install base
- Xbox Live service originated here
- NTSC 60Hz output

**Notable Facts:**

- Fastest-selling US console at launch (1.5M units in 3 weeks)
- Halo: Combat Evolved was the killer app
- Built-in HDD was revolutionary

### Japanese Xbox (2002)

**Distinguishing Features:**

- Launched February 22, 2002
- Limited success (530,000 units)
- Exclusive Japanese-developed titles
- Region-locked to Japanese games

**Notable Exclusives:**

- Metal Wolf Chaos (FROM Software)
- Phantom Dust (Japan Studio)
- Various visual novels and shooters
- Panzer Dragoon Orta (worldwide, but Japanese-developed)

### European Xbox (2002)

**Distinguishing Features:**

- Launched March 14, 2002
- PAL format (50/60Hz output varies)
- ~8 million units sold
- Strong UK market presence

## Hardware Revisions

### Version 1.0 (Launch)

| Feature         | Details                          |
| --------------- | -------------------------------- |
| **Motherboard** | Original design                  |
| **HDD**         | 8GB Western Digital or Seagate   |
| **DVD Drive**   | Thomson or Philips               |
| **Notes**       | Requires clock capacitor removal |

**Collector Note:** Version 1.0 has the infamous clock capacitor that can leak and damage the motherboard. Removal is strongly recommended.

### Version 1.1

| Feature         | Details                         |
| --------------- | ------------------------------- |
| **Motherboard** | Revised design                  |
| **HDD**         | 8GB various manufacturers       |
| **DVD Drive**   | Thomson, Philips, or Samsung    |
| **Notes**       | Still has clock capacitor issue |

### Version 1.2-1.4

| Feature         | Details                           |
| --------------- | --------------------------------- |
| **Motherboard** | Cost-reduced designs              |
| **HDD**         | 8GB or 10GB                       |
| **DVD Drive**   | Multiple manufacturers            |
| **Notes**       | v1.6 has integrated video encoder |

### Version 1.6 (Final)

| Feature         | Details                           |
| --------------- | --------------------------------- |
| **Motherboard** | Final cost-reduced design         |
| **HDD**         | 8GB Seagate typically             |
| **Video**       | Xcalibur integrated video encoder |
| **Notes**       | No clock capacitor issue          |

**Collector Note:** v1.6 is safe from capacitor issues but hardest to mod.

## Collecting Tips

### What to Look For

**Console Condition:**

- Check for clock capacitor damage (v1.0-1.4)
- Test DVD drive (common failure point)
- Verify HDD health
- Check controller ports for wear
- Listen for unusual fan or HDD sounds

**Game Condition:**

- DVD condition is paramount
- Check manual and inserts
- Original case artwork
- No rental stickers or damage

### Red Flags

**Common Issues:**

- Clock capacitor leakage (corrosion on board)
- Failed DVD drives (especially Thomson)
- Dying hard drives (errors, slow loading)
- Broken controller breakaway cables
- Stuck/scratching disc trays

**Fake/Reproduction:**

- Xbox games are rarely faked due to low values
- Focus on complete sets and condition
- Steel Battalion controllers are occasionally reproduced

### Storage & Preservation

**Console Care:**

- Remove clock capacitor if v1.0-1.4
- Keep in horizontal position
- Ensure good ventilation
- Clean disc drive regularly
- Consider HDD replacement/upgrade (modded units)

**Game Care:**

- Store discs in cases
- DVD rot can occur - check valuable discs
- Clean with microfiber, center to edge
- Resurface scratched discs if needed
