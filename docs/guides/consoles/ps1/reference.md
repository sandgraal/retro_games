# PlayStation 1 Reference Guide

_Last updated: December 2025_

The original PlayStation revolutionized gaming by making 3D graphics accessible, embracing CD-ROM technology, and courting mature audiences and third-party developers. Sony's first console established them as a gaming powerhouse and defined the fifth generation.

## Technical Specifications

### CPU

| Component        | Specification                        |
| ---------------- | ------------------------------------ |
| **Processor**    | MIPS R3000A                          |
| **Clock Speed**  | 33.8688 MHz                          |
| **Architecture** | 32-bit RISC                          |
| **Cache**        | 4KB instruction                      |
| **Features**     | Geometry Transformation Engine (GTE) |

### Graphics (GPU)

| Component        | Specification                                          |
| ---------------- | ------------------------------------------------------ |
| **GPU Type**     | Custom Sony                                            |
| **Clock Speed**  | 53.2 MHz                                               |
| **Fill Rate**    | ~360,000 flat-shaded polygons/sec                      |
| **Texture Rate** | ~180,000 textured polygons/sec                         |
| **Color Depth**  | 16-bit (32,768 colors) or 24-bit                       |
| **Resolution**   | 256×224 to 640×480 (interlaced)                        |
| **VRAM**         | 1MB                                                    |
| **Features**     | Gouraud shading, texture mapping, no texture filtering |

### Memory

| Type           | Specification |
| -------------- | ------------- |
| **System RAM** | 2MB EDO DRAM  |
| **Video RAM**  | 1MB           |
| **Sound RAM**  | 512KB         |
| **CD Buffer**  | 32KB          |

### Audio (SPU)

| Component       | Specification                 |
| --------------- | ----------------------------- |
| **Sound Chip**  | Custom Sony SPU               |
| **Channels**    | 24 ADPCM                      |
| **Sample Rate** | Up to 44.1 kHz                |
| **Features**    | Digital reverb, ADSR envelope |
| **Output**      | Stereo, RCA or A/V multi-out  |

### CD-ROM Drive

| Component        | Specification    |
| ---------------- | ---------------- |
| **Speed**        | 2x (300 KB/s)    |
| **Format**       | CD-ROM, Audio CD |
| **Max Capacity** | 660MB per disc   |

| **Weight**            | 1.2 kg            |
| **Controller Ports**  | 2                 |

## Maintenance & Repair

### Common Issues

| Issue                   | Symptom                       | Solution                         |
| ----------------------- | ----------------------------- | -------------------------------- |
| Disc read errors        | Games won't load, skip        | Clean lens, adjust pot           |
| Overheating             | Shutdowns, graphical glitches | Clean vents, thermal paste       |
| Controller port failure | Controllers not detected      | Clean ports, check for bent pins |
| Video output issues     | No picture, distortion        | Check cables, capacitors         |
| Motor noise             | Loud spinning, grinding       | Lubricate spindle                |

### Cleaning Procedures

**Disc Lens Cleaning:**

1. Open console (void warranty sticker)
2. Use lens cleaning swab with 90%+ IPA
3. Gently wipe in circular motion
4. Let dry completely
5. Reassemble and test

**Laser Pot Adjustment:**

- Last resort after lens cleaning fails
- Use multimeter to measure resistance
- Adjust pot screw in tiny increments
- Document original position first
- Risk: too much adjustment = burned laser

**Disc Cleaning:**

1. Clean from center outward (radial)
2. Never clean in circles
3. Use microfiber cloth
4. Mild soap and water for stubborn dirt
5. Dry completely before use

### Capacitor Issues

Later-model PS1s and PSones may develop capacitor problems:

- Audio crackling or distortion
- Video flickering
- Unexpected shutdowns
- Solution: capacitor replacement (advanced repair)

- Use quality SCART cable

- S-Video is best stock option

## History & Legacy

### Development Timeline

| Year | Event                                                   |
| ---- | ------------------------------------------------------- |
| 1988 | Sony/Nintendo collaboration on "Play Station" CD add-on |
| 1991 | Partnership collapses, Sony continues independently     |
| 1993 | "PlayStation" (one word) development begins             |
| 1994 | Japanese launch (December 3)                            |
| 1995 | North American launch (September 9, $299)               |
| 1996 | European launch; Ridge Racer, Tekken establish platform |
| 1997 | Final Fantasy VII launches, RPG boom begins             |
| 1998 | Metal Gear Solid, DualShock controller debut            |
| 1999 | 50 million units sold; mature library                   |
| 2000 | PS2 launches, PSone redesign released                   |
| 2006 | Production officially ends (12 years!)                  |

### Cultural Impact

- **CD-ROM Era**: Proved optical media viable for games
- **Third-Party Focus**: Welcoming policies attracted Square, Enix, Konami
- **Mature Gaming**: Targeted older demographics, legitimized adult games
- **Memory Cards**: Standardized save game portability
- **Dual Analog/Rumble**: DualShock became industry standard
- **PlayStation Brand**: Established Sony as gaming giant

### Why It Won

1. **Price**: Undercut Saturn at $299 vs $399
2. **Developer Support**: Easier to develop for than Saturn
3. **Third-Party Games**: Secured Final Fantasy, Tomb Raider
4. **Marketing**: "U R Not E" campaign targeted teens/adults
5. **Library Depth**: Massive, diverse game catalog

## Quick Reference Card

**Console Facts:**

- Released: December 3, 1994 (JP) / September 9, 1995 (NA)
- Discontinued: 2006
- Units sold: 102.49 million
- Library: 7,918 total worldwide titles

**Key Specifications:**

- CPU: 33.8 MHz MIPS R3000A

**Collecting Priorities:**

2. DualShock controller(s)
