## Console Overview

### Introduction

The TurboGrafx-16 (known as PC Engine in Japan) was a groundbreaking console that bridged the 8-bit and 16-bit eras. Despite its "16-bit" marketing, it featured an 8-bit CPU with dual 16-bit graphics processors, delivering visuals that rivaled early 16-bit systems while launching two years before the Genesis and three years before the SNES.

### Historical Significance

**Industry Impact:**

- First console with CD-ROM addon (1988)
- Pioneered portable console gaming (TurboExpress)
- Created the "shooter paradise" library
- Introduced the HuCard format (credit card-sized games)
- Smallest mainstream console of its generation
- First console to receive full anime cutscenes (via CD-ROM²)

**Regional Disparity:**

- Massive success in Japan (#1 console 1987-1990)
- Commercial failure in North America
- Never released in Europe (officially)
- Resulted in severely truncated NA library

## Technical Specifications

### Processor Details

**CPU: HuC6280**

- Modified 65C02 core @ 7.16 MHz
- 8-bit data bus, 16-bit address bus
- 21-bit address space (2 MB max)
- Hardware timer and I/O controller
- Built-in PSG sound generation

**Note:** Marketing as "16-bit" referenced the dual 16-bit graphics processors, not the CPU.

### Graphics Architecture

**HuC6270 (VDC - Video Display Controller):**

- Background layer: up to 1024 × 512 virtual screen
- 64 sprites on screen (16 per scanline)
- Sprite sizes: 16×16 to 32×64 pixels
- Scroll: horizontal and vertical

**HuC6260 (VCE - Video Color Encoder):**

- 512-color palette
- All 512 simultaneously available
- Video signal generation
- Composite and S-Video encoding

### Resolution Modes

| Mode     | Resolution | Notes               |
| -------- | ---------- | ------------------- |
| Standard | 256 × 239  | Most common         |
| Hi-Res   | 512 × 242  | Used in some games  |
| Low-Res  | 256 × 224  | Overscan compatible |

### Sound System

| Component    | Capability                     |
| ------------ | ------------------------------ |
| PSG Channels | 6 channels wavetable synthesis |
| Waveform     | 32 sample, 5-bit depth         |
| Noise        | 1 channel dedicated noise      |
| Stereo       | Full left/right panning        |
| CD Audio     | 16-bit 44.1 kHz (CD-ROM games) |
| ADPCM        | Sampling support (CD-ROM²)     |

## Common Issues & Repairs

### TurboGrafx-16/PC Engine

**Capacitor Failure:**

- Most common issue (age-related)
- Symptoms: no video, distorted audio, won't power on
- Solution: Full capacitor replacement ($20-40 parts)
- Difficulty: Moderate (soldering required)

**HuCard Slot Issues:**

- Dirty contacts prevent game reading
- Solution: Clean with isopropyl alcohol
- Prevention: Regular maintenance

**Power Supply:**

- Original PSUs fail or provide inconsistent power
- Solution: Modern replacement PSU ($20-30)

### TurboExpress/PC Engine GT

**Critical: Screen Failure**

- Capacitor leakage damages screen
- Most units have dead or dying screens
- Solution: LCD replacement mod ($80-150)
- Prevention: Recap before failure

**Capacitor Replacement Required:**

- Nearly all units need recapping
- Sound, video, and power affected
- Professional service recommended

### CD-ROM Systems

**Laser Pickup Failure:**

- Age-related failure (25+ years old)
- Symptoms: Disc not reading, audio skipping
- Solution: Laser replacement or adjustment
- Parts becoming scarce

**Mechanical Gear Failure:**

- Plastic gears wear or crack
- Symptoms: Tray not opening/closing
- Solution: Gear replacement kit

## Region Modifications

### Playing Japanese Games on TurboGrafx-16

**Physical Modification:**

1. Pin modification (remove pin 29 connection)
2. HuCard adapter purchase
3. Region-free mod board

**Recommended:** HuCard adapter (non-destructive, ~$20-40)

### Playing NA Games on PC Engine

**Physical Modification:**

1. TurboChip adapter
2. Pin modification
3. Both more complex than JP→NA

### CD Region

CD-ROM games are **region-free** on all systems. Any CD game plays on any CD-equipped system.

## Collecting Tips

### Entry Points

**Budget Start ($150-250):**

- Japanese PC Engine console
- Common HuCard games (shooters, platformers)
- Focus on Japanese library (much cheaper)

**Standard Start ($400-600):**

- TurboGrafx-16 console
- 5-10 common NA games
- TurboTap for multiplayer

**Comprehensive Start ($800+):**

- TurboDuo or PC Engine Duo
- CD games access
- Key HuCard and CD titles

### What to Prioritize

**Console Priority:**

1. Working console with video output
2. TurboTap/Multitap for multiplayer
3. CD capability (Duo or addon)
4. Backup device (saves)

**Game Priority:**

1. Shooters (genre specialty)
2. Essential action titles
3. CD RPGs (Ys, etc.)
4. Deep cuts for enthusiasts

### Import Considerations

**Advantages of Japanese Library:**

- 650+ games vs ~143 NA games
- Much lower prices on common titles
- Access to exclusive classics
- CD games region-free anyway

**Challenges:**

- Text-heavy games require Japanese
- Different connector shape (adapter needed)
- Some voltage considerations
