## Console Overview

### Introduction

The Nintendo Wii revolutionized gaming with motion controls and an approachable design that attracted both hardcore gamers and entirely new audiences. Its unique Wii Remote controller, affordable price point, and innovative software made it the best-selling home console of its generation despite being significantly less powerful than competitors.

### Historical Significance

**Industry Impact:**

- Pioneered mainstream motion-controlled gaming
- Demonstrated casual gaming market potential
- Outsold PS3 and Xbox 360 in its generation
- Virtual Console brought retro gaming to mainstream
- Wii Sports became cultural phenomenon

**Design Philosophy:**

- "Blue Ocean" strategy targeting non-gamers
- Lower power consumption than competitors
- Backward compatible with GameCube
- Focus on local multiplayer and party games
- Innovative over graphically powerful

## Technical Specifications

### CPU - IBM "Broadway"

- 32-bit IBM PowerPC processor
- 729 MHz clock speed
- 90nm SOI process
- Based on GameCube "Gekko" architecture
- Enhanced version for better performance

### GPU - ATI "Hollywood"

- 243 MHz core clock
- Based on GameCube "Flipper" GPU
- 3 MB embedded 1T-SRAM
- Texture cache and vertex/pixel shaders
- Maximum resolution: 480p (not HD)

### Memory

| Type  | Size   | Purpose                             |
| ----- | ------ | ----------------------------------- |
| MEM1  | 24 MB  | Main system RAM (1T-SRAM)           |
| MEM2  | 64 MB  | Extended RAM (GDDR3)                |
| Flash | 512 MB | Internal storage for saves/channels |

### Storage Options

| Medium               | Capacity    | Notes                         |
| -------------------- | ----------- | ----------------------------- |
| Internal Flash       | 512 MB      | Saves, channels, WiiWare      |
| SD Card              | Up to 32 GB | SDHC support added via update |
| USB Storage          | Up to 2 TB  | Limited functionality         |
| GameCube Memory Card | 4-64 MB     | For GCN saves only            |

### Video Output

| Output    | Resolution | Cable Type                   |
| --------- | ---------- | ---------------------------- |
| Composite | 480i       | Standard AV cable (included) |
| S-Video   | 480i       | Optional accessory           |
| Component | 480p/480i  | Optional, best quality       |
| RGB SCART | 480i       | PAL regions only             |

**Note:** The Wii does not support HD output natively. Third-party HDMI adapters upscale to 480p maximum native resolution.

### Audio

- Stereo audio via AV cables
- Dolby Pro Logic II surround encoding
- 16-bit 48 kHz sample rate
- Speaker in Wii Remote for effects

## Regional Differences

### NTSC vs PAL

| Feature          | NTSC (NA/JP) | PAL (EU/AU)   |
| ---------------- | ------------ | ------------- |
| Refresh Rate     | 60 Hz        | 50 Hz default |
| Resolution       | 480i/480p    | 576i/480p     |
| Component Output | Yes          | Yes           |
| RGB SCART        | No           | Yes           |

### Region Locking

The Wii is region-locked. Games must match console region:

- **NTSC-U**: United States, Canada, Latin America
- **NTSC-J**: Japan
- **PAL**: Europe, Australia, most of Asia

**Bypass Options:**

- Homebrew Channel (software mod)
- Modchips (hardware mod, less common)
- Gecko region-free codes

## Controller Ecosystem

### Wii Remote (Wiimote)

**Original Wii Remote:**

| Feature        | Specification                 |
| -------------- | ----------------------------- |
| Motion Sensing | 3-axis accelerometer          |
| Pointer        | Infrared camera + sensor bar  |
| Buttons        | A, B, 1, 2, +, -, Home, Power |
| Connectivity   | Bluetooth                     |
| Power          | 2× AA batteries               |
| Speaker        | Built-in for game effects     |
| Rumble         | Yes                           |

**Wii Remote Plus:**

- Integrated MotionPlus (enhanced motion)
- Required for some later games
- Slightly heavier than original

### Wii MotionPlus

- Accessory for original Wii Remote
- Adds gyroscope for precise motion
- Required: Wii Sports Resort, Skyward Sword, etc.

### Nunchuk

- Analog stick + 2 buttons (C, Z)
- Connects to Wii Remote
- 3-axis accelerometer
- Required for many games

### Classic Controller / Pro

| Variant     | Features                         |
| ----------- | -------------------------------- |
| Classic     | Tethered to Wiimote, SNES-style  |
| Classic Pro | Grips added, improved ergonomics |
| Third-party | Various wireless options         |

### GameCube Controller Support

- Original RVL-001 only
- 4 controller ports
- Required for some games (Smash Bros. preference)
- Full backward compatibility

## Maintenance and Care

### Common Issues

| Issue                  | Cause                | Solution                    |
| ---------------------- | -------------------- | --------------------------- |
| Disc read errors       | Dirty/damaged lens   | Lens cleaning, pot adjust   |
| Black screen           | Capacitor issues     | Capacitor replacement       |
| No power               | Power supply failure | Replace PSU                 |
| Controller sync issues | Bluetooth board      | Resync or board replacement |
| Sensor bar not working | LED failure          | Replace sensor bar          |

### Cleaning Guide

**Console:**

1. Unplug and power off completely
2. Use compressed air for vents
3. Wipe exterior with microfiber cloth
4. Avoid moisture near disc slot

**Controllers:**

1. Remove batteries
2. Clean button gaps with toothpick
3. Wipe with slightly damp cloth
4. Clean IR sensor window gently

**Game Discs:**

1. Wipe from center outward (never circular)
2. Use microfiber cloth
3. Mild soap and water for stubborn marks
4. Dry completely before use

### Storage Recommendations

- Store horizontally (reduces disc drive stress)
- Keep in ventilated area
- Away from heat sources
- Remove batteries from controllers when storing

## Collecting Tips

### What to Look For

**Console Purchases:**

- Model number (RVL-001 for GCN BC)
- Working disc drive
- Included Wiimote + Nunchuk
- Sensor bar
- Power adapter (region-specific)

**Game Purchases:**

- CIB preferred for Nintendo exclusives
- Disc condition (Wii discs scratch easily)
- Manual included
- Original case (not replacement)

### Price Factors

**Increases Value:**

- Complete in box with all inserts
- First-party Nintendo titles
- Limited print runs
- Sealed condition
- Special editions

**Decreases Value:**

- Loose disc only
- Heavy disc scratches
- Missing manual
- Damaged case
- Water/smoke damage
