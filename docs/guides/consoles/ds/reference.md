## Technical Specifications

### Processors

| Component        | Specification                             |
| ---------------- | ----------------------------------------- |
| **Main CPU**     | ARM946E-S @ 67 MHz                        |
| **Co-processor** | ARM7TDMI @ 33 MHz                         |
| **Architecture** | Dual 32-bit ARM                           |
| **Purpose**      | Main handles 3D, co-proc handles 2D/audio |

### Memory

| Type          | Specification   |
| ------------- | --------------- |
| **Main RAM**  | 4MB             |
| **ARM7 RAM**  | 64KB            |
| **VRAM**      | 656KB           |
| **Cartridge** | Up to 512MB ROM |

### Graphics

| Component      | Specification                      |
| -------------- | ---------------------------------- |
| **Resolution** | 256 × 192 per screen               |
| **Colors**     | 262,144 (18-bit)                   |
| **3D Engine**  | Custom, 120,000 polygons/sec       |
| **2D Engine**  | Hardware sprite/BG, GBA compatible |
| **Features**   | Cel-shading, fog, edge marking     |

### Audio

| Component      | Specification                   |
| -------------- | ------------------------------- |
| **Channels**   | 16 simultaneous                 |
| **Format**     | ADPCM, PCM                      |
| **Microphone** | Built-in                        |
| **Output**     | Stereo speakers, headphone jack |

### Connectivity

| Feature          | Specification                       |
| ---------------- | ----------------------------------- |
| **Wi-Fi**        | 802.11b (WEP only on original)      |
| **Local**        | Wireless multiplayer, Download Play |
| **GBA Slot**     | Plays GBA games (DS/DS Lite only)   |
| **Nintendo WFC** | Online service (discontinued 2014)  |

## Hardware Variants

### Nintendo DS (NTR-001) - 2004

**Original "Phat" Model:**

| Feature      | Details                         |
| ------------ | ------------------------------- |
| **Size**     | 148.7 × 84.7 × 28.9 mm          |
| **Weight**   | 275g                            |
| **Screens**  | Dual 3" TFT LCD                 |
| **Battery**  | 6-10 hours (lithium-ion)        |
| **GBA Slot** | Yes                             |
| **Colors**   | Silver, electric blue, graphite |

**Collector Notes:**

- First DS model, nostalgic
- Chunky design, less pocket-friendly
- GBA backwards compatibility
- Less common now than DS Lite

### Nintendo DS Lite (USG-001) - 2006

**Slimmed Redesign:**

| Feature        | Details                    |
| -------------- | -------------------------- |
| **Size**       | 133 × 73.9 × 21.5 mm       |
| **Weight**     | 218g                       |
| **Screens**    | Dual 3" TFT LCD (brighter) |
| **Battery**    | 15-19 hours                |
| **GBA Slot**   | Yes (protrudes slightly)   |
| **Brightness** | 4-level adjustment         |

**Collector Notes:**

- **Most popular DS model**
- Best balance of features and portability
- Many color variations and special editions
- GBA slot retained

### Nintendo DSi (TWL-001) - 2008

**Digital Focus Update:**

| Feature      | Details                         |
| ------------ | ------------------------------- |
| **Size**     | 137 × 74.9 × 18.9 mm            |
| **Weight**   | 214g                            |
| **Screens**  | 3.25" TFT LCD (slightly larger) |
| **Cameras**  | 2 (inner and outer) 0.3MP       |
| **GBA Slot** | **REMOVED**                     |
| **Storage**  | SD card slot, internal 256MB    |
| **DSi Shop** | Digital download store          |

**Collector Notes:**

- No GBA backwards compatibility
- Camera features
- DSi-exclusive games exist
- DSiWare downloads (shop closed 2017)

### Nintendo DSi XL (UTL-001) - 2009

**Large Screen Variant:**

| Feature     | Details                        |
| ----------- | ------------------------------ |
| **Size**    | 161.4 × 91.4 × 21.2 mm         |
| **Weight**  | 314g                           |
| **Screens** | 4.2" TFT LCD                   |
| **Battery** | 13-17 hours                    |
| **Target**  | Larger screens for readability |

**Collector Notes:**

- Best for accessibility
- Limited color options
- Same DSi features (no GBA slot)
- More visible screens

## Collecting Considerations

### Which Model?

| Goal                 | Recommendation                  |
| -------------------- | ------------------------------- |
| Play GBA + DS        | DS Lite (best balance)          |
| DS Only              | DSi or DSi XL                   |
| DSiWare preservation | DSi/DSi XL (download while can) |
| Special editions     | DS Lite has most options        |
| Large screens        | DSi XL                          |

### Common Issues

| Issue               | Affected Models | Solution            |
| ------------------- | --------------- | ------------------- |
| Hinge cracks        | DS Lite         | Shell replacement   |
| Screen scratches    | All             | Screen protectors   |
| Touchscreen drift   | Older units     | Calibration/replace |
| Battery degradation | All             | Battery replacement |
| Dead pixels         | All             | None (accept it)    |

### Fake Game Detection

DS fakes are less common than GBA but exist:

**Signs of Fake:**

- Wrong cart color
- Poor label quality
- No serial number on label back
- Light weight
- Wrong Nintendo logo

**Commonly Faked:**

- Pokemon games (all versions)
- Dragon Quest games
- Expensive RPGs
