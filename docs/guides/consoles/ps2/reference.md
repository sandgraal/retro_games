## Regional Variants

### SCPH-10000 (Japan Launch, 2000)

**Form Factor:** Original "fat" design, black finish.

**Key Differences:**

- First production model
- Required external HDD adapter
- DVD playback required separate purchase
- PCMCIA slot for HDD connectivity

**Collector Notes:**

- Earliest production, some reliability concerns
- Rare for collectors outside Japan
- Some units have blue disc tray

### SCPH-30000 Series (2000-2003)

**Form Factor:** Refined "fat" design, multiple colors.

**Key Features:**

- Most common early model
- DVD playback integrated
- Internal expansion bay
- Versions: 30001 (NA), 30002/3/4 (EU), etc.

**Collector Notes:**

- Reliable model, good for collecting
- HDD adapter support (30001)
- Various special editions exist

### SCPH-50000 Series (2003-2004)

**Form Factor:** "Fat" design with internal improvements.

**Key Features:**

- Integrated infrared sensor
- More reliable laser assembly
- Quieter operation
- HDD adapter support continues

**Collector Notes:**

- Considered most reliable "fat" model
- Good choice for primary gaming system
- Some limited editions

### SCPH-70000 Series (Slim, 2004-2007)

**Form Factor:** Dramatically smaller "slimline" design.

**Key Features:**

- 75% smaller and 75% lighter than original
- External power supply (AC adapter)
- No internal expansion bay (no HDD support)
- Built-in ethernet (most variants)
- New disc loading mechanism

**Collector Notes:**

- Cannot use official HDD
- More portable, less expandable
- Various color options released

### SCPH-90000 Series (Slim, 2007-2013)

**Form Factor:** Final slim revision.

**Key Features:**

- Further cost reductions
- Same slim form factor
- Final production models
- Some reliability improvements

**Collector Notes:**

- Last PS2 models produced
- Good for general use
- Less collectible than special editions

## Technical Specifications

### CPU: Emotion Engine (EE)

- **Architecture:** Custom 128-bit CPU
- **Clock Speed:** 294.912 MHz
- **Components:**
  - MIPS R5900 core (64-bit)
  - 2× Vector Processing Units (VPU0/VPU1)
  - 10 DMA channels
- **Memory:** 32MB RDRAM

### Graphics Synthesizer (GS)

- **Clock Speed:** 147.456 MHz
- **Fill Rate:** 2.4 billion pixels/second
- **VRAM:** 4MB embedded DRAM
- **Features:**
  - Hardware anti-aliasing
  - MPEG2 decoding
  - Multi-texturing
- **Output:** 480i/480p, PAL modes, component output

### Audio: SPU2

- **Channels:** 48 voices
- **Memory:** 2MB audio RAM
- **Features:**
  - ADPCM playback
  - Hardware reverb
  - 3D audio processing

### I/O Processor (IOP)

- **CPU:** PlayStation 1 CPU (PSX IOP)
- **Purpose:** PS1 backwards compatibility, I/O handling
- **Clock:** 36.864 MHz

## Media & Compatibility

### Disc Formats

| Format     | Supported | Notes                        |
| ---------- | --------- | ---------------------------- |
| PS2 DVD    | Yes       | Primary game format          |
| PS2 CD-ROM | Yes       | Some budget/early titles     |
| PS1 Games  | Yes       | Full backwards compatibility |
| DVD-Video  | Yes       | Movie playback               |
| CD Audio   | Yes       | Music playback               |

### Region Locking

- **Strict region enforcement** for both PS2 and PS1 games
- **Regions:** NTSC-U/C (Americas), NTSC-J (Japan/Asia), PAL (Europe/Australia)
- **Bypass methods:** Swap Magic, Free McBoot, modchip

### Backwards Compatibility

- **PS1 games:** Nearly 100% compatible
- **Memory cards:** PS1 cards readable via adapter
- **Controllers:** PS1 games work with DualShock 2

## Collecting Guide Quick Reference

### Condition Grading

| Grade     | Case | Manual | Disc | Expected Value |
| --------- | ---- | ------ | ---- | -------------- |
| Sealed    | Mint | N/A    | N/A  | 200-400% CIB   |
| CIB Mint  | 9-10 | 9-10   | 9-10 | 100% baseline  |
| CIB Good  | 7-8  | 7-8    | 8-9  | 70-85% CIB     |
| Complete  | 5-6  | 5-6    | 7-8  | 50-70% CIB     |
| Disc Only | N/A  | N/A    | 7-9  | 30-50% CIB     |

### What to Check When Buying

1. **Disc condition:** PS2 DVDs more scratch-sensitive than CDs
2. **Laser wear:** Test multiple disc types (DVD and CD games)
3. **Controller ports:** Check for loose connections
4. **Read errors:** Common on older fat models
5. **Case/manual:** Black cases show wear, manuals often missing

### Storage Recommendations

- Store games vertically in cases
- Keep away from heat sources
- Avoid stacking heavy items on disc cases
- Use protective sleeves for rare titles
- Climate control for long-term storage

## Maintenance & Repair

### Common Issues

| Problem                 | Cause                       | Solution                            |
| ----------------------- | --------------------------- | ----------------------------------- |
| Disc Read Error (DRE)   | Worn/dirty laser            | Clean or adjust laser, replace      |
| No disc spin            | Motor failure               | Replace spindle motor               |
| Controller not detected | Port damage or dirty        | Clean ports, check ribbon cables    |
| No video output         | Bad AV cable or chip        | Test cables, may need repair        |
| Overheating             | Dust buildup, thermal paste | Clean system, reapply thermal paste |
| Memory card not reading | Dirty slots                 | Clean slots with isopropyl alcohol  |

### Laser Adjustment

**For DRE issues on fat models:**

1. Open console (warranty void)
2. Locate white gear with adjustment pot
3. Turn gear slightly (1/8 turn) clockwise
4. Test with DVD and CD games
5. May need multiple small adjustments

**Warning:** Over-adjustment can burn out laser faster.

### Cleaning Guide

**Exterior:**

1. Compressed air for vents and ports
2. Damp microfiber cloth for surfaces
3. Isopropyl alcohol for stubborn marks

**Interior (fat models):**

1. Remove cover (8 screws)
2. Blow out dust with compressed air
3. Clean laser lens with lens cleaner disc or cotton swab
4. Check thermal paste on older units

## Modding & Homebrew

### Free McBoot (FMCB)

**Purpose:** Softmod via memory card, no hardware modification.

**Features:**

- Boot homebrew from memory card
- Load games from USB/HDD (OPL)
- Region-free capability
- Cheats and game enhancement

**Compatibility:** Works on Slim 7000x and below, all fat models

### Open PS2 Loader (OPL)

**Purpose:** Load games from USB, network, or HDD.

**Benefits:**

- Preserve original discs
- Faster load times (HDD)
- Convenience for large collections

### Hard Drive Gaming

**Requirements:**

- Fat PS2 with Network Adapter
- IDE hard drive (40GB+) or SATA adapter
- FMCB memory card
- OPL software

**Advantages:**

- Fastest loading method
- Organize large game libraries
- Preserve disc-based games

## Emulation & Preservation

### Recommended Emulators

| Emulator  | Platform       | Accuracy | Notes                       |
| --------- | -------------- | -------- | --------------------------- |
| PCSX2     | Win/Mac/Linux  | High     | Most compatible, active dev |
| AetherSX2 | Android        | High     | Mobile PCSX2 fork           |
| Play!     | Multi-platform | Medium   | Alternative option          |

### Requirements

**Minimum for PCSX2:**

- CPU: Quad-core 3.2GHz+
- GPU: DirectX 11/OpenGL 4.5
- RAM: 4GB+
- BIOS files required

### FPGA Status

- No consumer FPGA PS2 solution exists (2025)
- Complexity of Emotion Engine prevents FPGA recreation
- MiSTer does not support PS2
