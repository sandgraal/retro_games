# Collecting Fighting Games: The Complete Genre Guide

_Last updated: December 2025_

## Introduction

Fighting games represent gaming's purest competitive form - two players, skill versus skill, with decades of evolution in mechanics and meta. From arcade cabinets to global esports, the genre has produced some of gaming's most beloved franchises.

**What Defines Fighting Games?**

- One-on-one (or team) combat
- Health bars and round systems
- Complex input mechanics
- Competitive depth and mastery
- Strong arcade heritage

**Sub-genres Covered:**

- 2D Traditional Fighters
- 3D Fighters
- Anime/Air-Dashers
- Platform Fighters
- Arena Fighters
- Tag Team Fighters

## Collecting Tips

### What to Prioritize

**High Value Targets:**

- Neo Geo AES carts (limited production)
- Dreamcast fighters (limited print runs)
- Japanese Saturn imports
- Limited editions with figures
- CPS2/3 boards

**Best Value Picks:**

- PS2 fighters (undervalued)
- 360/PS3 generation (very cheap)
- Greatest Hits versions for play
- Digital re-releases

### Regional Considerations

**Import-Worthy:**

- Japan often gets complete editions first
- Some games never localized
- Neo Geo games region-free
- PS1/PS2 Japanese versions may differ

**Notable Japan Exclusives:**

| Title               | Platform   | Notes                |
| ------------------- | ---------- | -------------------- |
| Tobal 2             | PS1        | 200 characters       |
| Asuka 120% series   | Saturn/PS1 | School fighters      |
| Various KoF entries | Neo Geo    | Earlier releases     |
| Melty Blood         | PS2        | Visual novel fighter |
| Battle Fantasia     | Arcade     | Earlier than console |

### Condition Matters

| Factor          | Impact                        |
| --------------- | ----------------------------- |
| Disc scratches  | Fighting games heavily played |
| Case wear       | Standard for genre            |
| Manual present  | Move lists valuable           |
| Original vs. GH | Collector preference          |
