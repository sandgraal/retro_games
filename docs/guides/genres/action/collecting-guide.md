# Collecting Action Games: The Complete Genre Guide

_Last updated: December 2025_

## Introduction

The action genre is gaming's broadest category, encompassing everything from side-scrolling beat 'em ups to 3D hack-and-slash epics. This guide focuses on character-action games, beat 'em ups, and action-adventure titles - the games where reflexes and combat skill define the experience.

**What Defines Action Games?**

- Real-time combat and movement
- Player skill over stats/grinding
- Reflexes, timing, and pattern recognition
- Often arcade-influenced design

**Sub-genres Covered:**

- Beat 'em ups / Brawlers
- Hack and Slash
- Character Action (Stylish Action)
- Action-Adventure
- Arcade Action

## Collecting Tips

### What to Prioritize

**High Value Targets:**

- Treasure games (Gunstar Heroes, Guardian Heroes)
- Late-life PS2 releases (Godhand)
- First-party Nintendo action titles
- PlatinumGames limited prints
- Saturn beat 'em ups

**Best Value Picks:**

- PS3 action games (very cheap)
- Common PS2 action titles
- Xbox 360 character action
- Digital re-releases

### Condition Matters

| Factor            | Impact                            |
| ----------------- | --------------------------------- |
| Disc scratches    | Action games often played heavily |
| Case wear         | Standard expectation              |
| Manual present    | Valuable for PS2/Xbox era         |
| Original (not GH) | Significant for collectors        |

### Regional Considerations

**Import-Worthy:**

- Japan often has exclusive action games
- Some games censored in West
- Region-free consoles help

**Japan Exclusives Worth Importing:**

| Title                    | Platform | Notes                |
| ------------------------ | -------- | -------------------- |
| Onimusha: Dawn of Dreams | PS2      | JP voice option      |
| Samurai Western          | PS2      | Full English         |
| Seven Samurai 20XX       | PS2      | English version rare |
| Various Musou games      | PS2/PS3  | Pre-localization     |
